# Completed Tasks

## ✅ Task 1: Hangman Game

### Description

A simple console-based Hangman game where the player guesses one letter at a time.

### Features

- Random word selection
- Maximum 6 incorrect guesses
- Displays guessed letters
- Win/Lose message
- Console-based interface

### Concepts Used

- Random Module
- Lists
- Strings
- Loops
- Conditional Statements

---

## ✅ Task 2: Stock Portfolio Tracker

### Description

A Python application that calculates the total investment value of selected stocks using a predefined dictionary of stock prices.

### Features

- User inputs stock name
- User inputs quantity
- Uses hardcoded stock prices
- Calculates total investment
- Saves output in a text file

### Concepts Used

- Dictionary
- User Input
- Arithmetic Operations
- File Handling

---

## ✅ Task 3: Task Automation

### Description

A Python automation script that automatically moves all JPG image files from one folder to another.

### Features

- Reads files from source folder
- Detects JPG images
- Moves files automatically
- Prevents manual work

### Concepts Used

- OS Module
- shutil Module
- File Handling
